# Photogram
A Simple Photo Sharing Application.

## Dependencies
- Sass
- NPM
- Apache
- PHP (Above 8.0)
- Node.js (Above 21.0.0)
- Composer

## Get Started

Install Composer Dependencies
```bash
composer install
```

Install NPM Dependencies
```bash
npm install
```

Run Development Server to Serve Static Files
```bash
npm run dev
```

Compile and build Production Files
```bash
npm run build
```
